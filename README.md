# Portfolio Optimization Using Markowitz Mean-Variance Theory

## Project Description
This project constructs an optimal investment portfolio using Markowitz Mean-Variance Optimization.

## Research Question
How can investors maximize returns while minimizing portfolio risk?

## Dataset
Historical stock prices obtained from Yahoo Finance.

## Methodology
- Data Cleaning
- Return Calculation
- Covariance Matrix
- Efficient Frontier
- Portfolio Optimization

## Software Used
- Python
- Stata
- Excel

## Results
The efficient frontier was generated and optimal portfolio weights were estimated.
